## Description

Travel app that allows users to manage itineraries when visiting new cities.
<br>

## Data

Itinerary

```json
{
  "location_name": String,
  "latitude": Number,
  "longitude": Number,
  "address": String,
  "description": String,
  "activities": Array,
}

```

Activities

```json
{
  "activityId": Number,
  "activity_name": String,
}

```
